Como convertirse en Desarrollador Front-end

Para hoy para convertirse en un desarrollador front-end al día de hoy que quisiera libremente se esfuerzan por seguir el proceso descrito a continuación

  • Aprender HTML
  • Aprender CSS
  • Aprender JavaScript
  • Aprender DOM
  • Aprender JSON y APIs
  • Aprender fundamentos de UI
  • Aprender CLI
  • Aprender ingeniería de software (Control de versiones, testeo, monitoreo, calidad de código, metodologías de desarrollo, diseño de aplicaciones, etc)
  • Aprender a extender tu set de herramientas de desarrollo (Angular, React, Vue, Webpack, etc)
  • Aprender Node.js

Por el momento este curso se centrará en los tres primeros puntos, que es lo básico.
